# Digicrafts Tap

## How do I install these formulae?

`brew install digicrafts/tap/<formula>`

Or `brew tap digicrafts/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "digicrafts/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
